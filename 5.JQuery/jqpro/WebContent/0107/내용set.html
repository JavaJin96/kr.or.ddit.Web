내용 변경/설정

#result1.html(res) : setter(res의 내용으로 result1을 변경해라.)
res의 내용으로 result1을 출력해라.
res의 내용 속에 html태그가 있을 경우(innerHTML과 같다. / 포함되지 않아도 가능)
#result1.text(res) : setter(res의 내용으로 result1을 변경해라.)
res의 내용으로 result1을 출력해라.
res의 내용 속에 html태그가 없을 경우(textContent와 같다. / 태그가 포함되면 문자열 자체로 출력된다.)
