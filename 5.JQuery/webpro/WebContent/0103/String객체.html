인덱스를 이용한 글자찾기

charAt(index) : index 위치의 글자를 리턴
charCodeAt(index) : index위치의 unicode값을 리턴

공백제거

trim() : 앞뒤의 공백을 제거한다.

데이터 입력체크

문자열 대체

replace(oldstr, newstr) : oldstr을 newstr로 바꿔라.
textarea에 입력하여 result4에 출력

문자열 추출

substring(start, end) : start에서 end-1까지 문자열을 추출한다.
slice(start, end) : start에서 end-1까지 문자열을 추출한다.
substr(start, length) : start에서 length의 갯수만큼 추출

문자열 분리

split(구분자, 갯수) : 구분자를 기준으로 문자열을 갯수만큼 분리한다.
결과는 배열로 리턴한다.
