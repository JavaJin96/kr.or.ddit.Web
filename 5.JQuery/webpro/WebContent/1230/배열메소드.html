요소 찾기

-indexOf(item, start) : 맨 처음 만나는 item의 위치를 리턴한다.(start위치에서 시작 : 옵션임)
-lastIndexOf(item) : 맨 마지막에 만나는 item의 위치를 리턴한다.
'

추가 삭제

-push(a,b,c) : 배열의 마지막에 a,b,c 요소를 출력한다.
-pop() : 맨 마지막 요소를 삭제한다.
'

정렬

-sort : 배열의 요소를 정렬한다.

추출/ 수정

-slice(start, end) : start ~ end-1 사이의 요소를 추출해서 새로운 배열을 생성한다.
-splice(index, n , a, b, c) : idx번째에서 n개를 삭제하고 a,b,c를 삽입한다.

배열합치기 / 문자열변환

-a.concat(b) : a배열과 b배열을 합친다.
-a.join() : a배열을 문자열로 변환한다.
